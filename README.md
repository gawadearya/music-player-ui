# Music Player UI 🎵

A responsive Spotify-inspired Music Player User Interface built using HTML and CSS. This project recreates the look and feel of a modern music streaming platform with playlists, trending songs, navigation menus, and a music player section.

## Preview

This project features:
- Spotify-inspired user interface
- Responsive design
- Sidebar navigation
- Recently Played section
- Trending Music section
- Music Player Control Bar
- Modern and clean layout

## Technologies Used

- HTML5
- CSS3

## Getting Started

1. Clone the repository:

```bash
git clone https://github.com/gawadearya/music-player-ui.git
```

2. Open the project folder.

3. Run `index.html` in your browser.
